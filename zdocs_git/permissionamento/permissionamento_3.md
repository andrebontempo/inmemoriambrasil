

TRIBUTOS
Agora vamos trabalhar com o permissionamento.

Vamos começar primeiro e somente com o a página de Homenagens.

As regras são:

1 - Para enviar uma homenagem você deve estar logado.

2 - Para editar ou excluir tributos você deve ser o autor, dono do memorial, colaborador do memorial ou admin. Neste caso deve-se esconder os botões para quem não tem permissão.
